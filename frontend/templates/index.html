Election Tweets

These files contain Tweet collected from the Twitter streaming API.

These tweets were filtered using approximately 40 keywords related to Presidential candidates across both parties (primarily candidate names).

The summary files contain 10% of the tweets from the corresponding daily file.

The CSV files are generated from the daily JSON files, mimicking the effect of the parseTweets command found in the R streamR package.
